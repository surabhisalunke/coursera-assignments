TOP 3 PROGRAMMING LANGUAGES

PYTHON

  • Python is an interpreted,object-oriented, high-level programming language.
  • Python is simple,easy to learn.
  • Python supports modules and pakages, which encourages program modularity and code reuse
  • The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.
  • Debugging Python programs is easy: a bug or bad input will never cause a segmentation fault

JAVA

  • Java is a high-level programming language developed by Sun Microsystems.
  • It was originally designed for developing programs for set-top boxes and handheld devices, but later became a popular choice for creating web applications.
  • Java programs are interpreted by the Java Virtual Machine, or JVM, which runs on multiple platforms. This means all Java programs are multiplatform and can run on different platforms, including Macintosh, Windows, and Unix computers
